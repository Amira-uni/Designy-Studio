# Designy Studio – Portfolio & Services Website

## 🌐 Overview
Designy Studio is a personal portfolio and services website created to showcase my web development and design work. It serves as a central hub for my projects, skills, and services, helping potential clients understand what I offer in a clean and professional way.

## 🚀 Features
- Modern and responsive design
- Services section (web design, development, etc.)
- Portfolio/project showcase
- About section
- Contact section
- Mobile-friendly layout

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript

## 🎯 Purpose
This project was built to present my skills and services in a professional way and to attract freelance clients or job opportunities.
